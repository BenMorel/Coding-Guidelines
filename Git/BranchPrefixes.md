## Branch prefixes

| Abbrev.   | Reason                                                |
|-----------|-------------------------------------------------------|
| `feat`    | A feature branch                                      |
| `fix`     | A bugfix branch                                       |
| `hotfix`  | Hotfix changes for production issues                  |
| `cleanup` | Cleaning up / organizing the code                     |
| `poc`     | Branched out for some POC stuff that has to be pushed |
